Erweiterungen

Wie bereits im Ablauf beschrieben, wurde der bisherige Einstellungsdialog ersetzt. Statt nur eine einzelne Serveraddresse eingeben zu können, hat der User nun die Möglichkeit diverse Server zu verwalten. Zu jedem Server können jetzt Addresse, Benutzer und Passwort, die Projekte, deren Benutzer und die jeweiligen Kategorien und Berechtigungen verwaltet werden. Im wesentlichen entspricht die Funktionalität der des Web Interface.
