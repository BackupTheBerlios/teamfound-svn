Neuerungen am Web-Interface

Abbildung 2.1: Web-Interface Eingangsseite

Neuerungen ist gut, bisher gab es eigentlich noch kein Web-Interface :)

Die Version 0.2 des TeamFound Servers hatte noch die Option want=xml|html, über die man vom Server direkt eine im Browser darstellbare HTML-Seite anfordern konnte. Jedoch war dieser Teil des Servers zu grossen Teilen nicht implementiert.

In der aktuellen Version 0.3 setzen wir nur auf XML-Antworten, die dafür aber allesamt ein XSL-Transformations Stylesheet enthalten:

<?xml-stylesheet href=transform.xsl type=text/xsl?>

Über diese XSLT Datei können alle gängigen Browser das vom Server gelieferte XML selber in HTML umwandeln und anzeigen.
