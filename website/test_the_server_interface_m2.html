TeamFound - interface milestone 2 requests

These forms generate interface milestone 2 conform requests to test the server.

All requests are directed to

search

keyword =
category =the category-id (0 = no special category / top-category)
want =xml | html
version =interface version (Milestone 2 = 2)
command =server-command

add page

url =a complete url
category =the category-id (0 = no special category / top-category)
want =xml | html
version =interface version (Milestone 2 = 2)
command =server-command

get categories

want =xml | html
version =interface version (Milestone 2 = 2)
command =server-command

add category

name =the name of the new category
description =the description of the new category (optional)
subcategoryof=the parent-category-id (0 = top-category)
