Anfrage an Server

command=search
Der Kommando-Name

search.pl
Der (default) Skriptname der die Anfrage entgegennimmt (dann fällt das Argument command weg).
keyword=zzz
Eine durch ' ' (space) getrennte Liste der Suchwörter die alle vorkommen sollen (URL-Codiert mit ''%20'' getrennt, also eine Suche nach auto und tv wird als keyword=auto%20tv codiert). Dies beinhaltet eine serverseitig impliziete UND-Verknüpfung aller Suchworte. Fehlt der Parameter so soll ein Fehler zurückgegeben werden. Die genaue Definition einer umfassenderen Suchanfragen-Sprache (wie OR, NOT etc.) ist noch festzulegen, wird aber auf spaetere Interface-Milestones verschoben.

category=5&category=9&category=15
Eine Liste aller Kategorien-IDs, jeweils immer wieder neu mit category= durch die die Suche gemacht werden soll. Fehlt der Paramater, so soll alles durchsucht werden. (Grund: HTML-Formulare machen dies bei Listen und ComboBoxen mit Mehrfachauswahl genau so).
