Modell

Das Modell besteht aus der Klasse Category. Ein Category Objekt kann selber wieder Kategorie Objekte enthalten. Somit können die Kategorien vom Server abgeholt und im Speicher gehalten werden. Die Views können über die Eigenschaft Categories, des Controllers auf die Kategorien zugreifen und Anzeigen. Der Controller wiederum aktualisiert die Kategorien wenn es erforderlich ist.
