Anfrage an Server

command=addcategory
Der Kommando-Name

addcategory.pl
Der (default) Skriptname der die Anfrage entgegennimmt (dann fällt das Argument command weg).

name=kategoriename
Den Namen den die neue Kategorie bekommen soll

subcategoryof=45
Die Kategorie der die neue Kategorie untergeordnet werden soll. Soll eine neue 1st-Level Kategorie erstellt werden ist hier 0 (null) anzugeben.

description
Eine etwas längere Beschreibung der Kategorie (max. 255 Zeichen)
