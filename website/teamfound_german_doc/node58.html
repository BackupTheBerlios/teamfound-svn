Entwicklung der Band Objekte

Die Entwicklung dieser Komponenten basiert, wie ein sehr großer Teil der Windowsentwicklung auf COM. Das Component Object Model von Microsoft ist eines der ältesten Komponentensysteme und bildet zusammen mit dem Windows API die Grundlage für die Entwicklung von Windowssoftware. Da die Toolbar aus solchen Bandobjekten besteht, müssen also COM Objekte geschrieben werden.
