Anfrage an Server

command=addpage
Der Kommando-Name

addpage.pl
Der (default) Skriptname der die Anfrage entgegennimmt (dann fällt das Argument command weg).

category=453
Die Kategorien-Nummer zu der der Link hinzugefuegt werden soll, in dieser Version nur genau eine oder keine. Wird keine Kategorie angegeben, so soll die Top-Kategorie genommen werden (also nicht gefunden werden wenn bei der Suche eine genauere Kategorie angegeben wird).

url=http://yy.org/blabla.html
Die zu indizierende url, das Protokoll ist mit anzugeben
